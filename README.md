## Installation

You can install `tinytools` from
[Github](https://github.com/tidymass/tinytools).

``` r
if(!require(devtools)){
install.packages("devtools")
}
devtools::install_github("tidymass/tinytools")
```

``` r
library(tinytools)
```

``` r
tidymass::install_tinytools(from = "github", force = FALSE)
```
